# Data Not Included

## Notice

This repository was developed as part of the **Asian Development Bank (ADB) AI for Safer Roads Innovation Challenge 2026**.

The datasets used during development were provided under the Challenge's Data Use Agreement (FDUA) and are therefore **not included in this public repository**.

## Why the Data Is Not Available

The challenge data is subject to usage restrictions that prohibit public redistribution of the supplied datasets and certain challenge-derived outputs.

Accordingly, this repository does not contain:

- Original challenge datasets
- Road segment databases
- Traffic and speed observation data
- GeoJSON outputs derived from challenge data
- CSV outputs derived from challenge data
- Interactive maps containing challenge-derived segment information

## What Is Included

This repository contains:

- Methodology documentation
- Analysis notebook
- Explainable AI workflow
- Feature importance visualizations
- SHAP visualizations
- Project documentation
- Submission overview

These materials are provided for educational and demonstration purposes.

## Reproducibility

The analytical workflow and methodology are fully documented within the notebook and project documentation.

Researchers and practitioners may apply the same approach to their own road safety datasets, road inventories, speed observations, or open transportation data sources.

## Contact

For information regarding the challenge datasets, please refer to the official Asian Development Bank (ADB) AI for Safer Roads Innovation Challenge documentation and data access policies.
